# Bank Account Management System

A simple C++ console application that simulates basic banking operations using object-oriented programming (OOP).

## Features
- Create a bank account
- Deposit money
- Withdraw money (maintains a minimum balance of ₹500)
- Display account details
- Uses default and parameterized constructors

## Concepts Used
- Classes and Objects
- Constructors
- Encapsulation
- Member Functions
- Conditional Statements
- User Input/Output

## Technologies
- C++
- Standard Library (iostream)
